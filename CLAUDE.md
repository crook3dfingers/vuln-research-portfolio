# Portfolio Maintenance Instructions

This repo is the public index for `crook3dfingers/vuln-research-*`
campaign repositories.

When creating, researching, renaming, archiving, or materially updating any
vulnerability-research campaign, follow `TRACKING.md` and keep `README.md`
current.

Required invariants:

- Every campaign repo is named `vuln-research-<slug>`.
- Every campaign repo has the GitHub topic `vuln-research`.
- Every locally created campaign under `/home/ubuntu/vuln_research/projects`
  has a matching row in this portfolio when it becomes an active campaign.
- Public portfolio text must be disclosure-safe and must not include secrets,
  exploit recipes, private evidence, or vendor-sensitive details.
